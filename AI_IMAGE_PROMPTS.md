## Tips for Using These Prompts

### For Gemini
- Use the prompts as-is
- Add "educational illustration style" if needed
- Specify "simple background" for cleaner results
- Request "high resolution" for print quality

### For DALL-E / Midjourney
- Add style modifiers: "educational illustration", "simple design", "clean background"
- For symbols: "high contrast", "traditional Japanese style"
- For diagrams: "diagram style", "educational layout"

### Best Practices
- **Start simple** - Use the basic prompts first
- **Add details** - Refine prompts based on results
- **Be specific** - Mention "educational", "teaching materials", "clear"
- **Request variations** - Generate 2-3 versions to choose from
- **Specify colors** - For chakras, specify exact colors
- **Request high resolution** - For print quality (300 DPI equivalent)

## Image Specifications

### For Wisdom Cards
- **Size:** 750 x 1050 pixels
- **Resolution:** 300 DPI
- **Format:** PNG (supports transparency)
- **Style:** Simple, clear, educational

### For Teaching Materials
- **Size:** 1000 x 1000 pixels minimum
- **Resolution:** 300 DPI
- **Format:** PNG or JPG
- **Style:** Educational, clear, professional

### For Symbols
- **Size:** 1000 x 1000 pixels minimum
- **Background:** Transparent PNG preferred
- **Style:** High contrast, traditional accuracy
