## Detailed Steps

### Step 1: Download Gassho Images

1. **Open:** https://commons.wikimedia.org/wiki/Category:Gassho
2. **Browse:** Available images
3. **Select:** Images with public domain (CC0) license
4. **Click:** On image
5. **Download:** Click "More details" → "Original file" → Download
6. **Save as:** `gassho_1.png` in `01_Images/Gassho/` folder
7. **Repeat:** Download 2-4 more images
8. **Update:** `01_Images/Gassho/README.md` with attribution

### Step 2: Download Reiki Symbols

1. **Open:** https://commons.wikimedia.org/wiki/Category:Reiki_symbols
2. **Search:** For specific symbols:
   - Cho Ku Rei (Power symbol)
   - Sei He Ki (Harmony symbol)
   - Hon Sha Ze Sho Nen (Distance symbol)
   - Dai Ko Myo (Master symbol)
3. **Download:** Each symbol
4. **Save to:** `01_Images/Symbols/` folder
5. **Name:** `cho_ku_rei.png`, `sei_he_ki.png`, etc.
6. **Update:** `01_Images/Symbols/README.md` with attribution

### Step 3: Download Chakra Diagrams

1. **Open:** https://commons.wikimedia.org/wiki/Category:Chakras
2. **Browse:** Chakra diagrams
3. **Select:** Clear, educational diagrams
4. **Download:** 3-5 diagrams
5. **Save to:** `01_Images/Chakras/` folder
6. **Name:** `chakra_1.png`, `chakra_2.png`, etc.
7. **Update:** `01_Images/Chakras/README.md` with attribution

### Step 4: Add Attribution

For each image folder, update the README.md with:

```markdown
## Images

### Image 1: [filename].png
- **Source:** Wikimedia Commons (or Pixabay, etc.)
- **License:** Public Domain (CC0) or [license type]
- **URL:** [link to original]
- **Description:** [what the image shows]
```

### Step 5: Commit to GitHub

After downloading images:

```bash
cd RI-Reiki-Resources
git add .
git commit -m "Add [category] images"
git push
```

## Alternative: Pixabay Images

If Wikimedia Commons doesn't have what you need:

1. **Go to:** https://pixabay.com/images/search/[query]/
   - Gassho: https://pixabay.com/images/search/prayer%20hands/
   - Chakras: https://pixabay.com/images/search/chakra/
2. **Filter:** Free images only
3. **Download:** Click image → "Free Download" → Choose size
4. **Save:** To appropriate folder
5. **Note:** Pixabay images don't require attribution, but it's nice to credit

## Tips

- **Start with 3-5 images** per category
- **Choose high-resolution** images (at least 500x500 pixels)
- **Prefer PNG** format (supports transparency)
- **Check licenses** before downloading
- **Add attribution** to README files
