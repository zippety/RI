# Images: Visual Reiki Resources

Visual resources for teaching, learning, and practice.

## Folders

- **[Gassho/](Gassho/)** - Prayer position images
- **[Symbols/](Symbols/)** - Reiki symbol diagrams
- **[Chakras/](Chakras/)** - Chakra illustrations

## Image Sources

All images are from:
- Wikimedia Commons (public domain)
- Pixabay (free commercial use)
- Other free sources

**Always check the license file in each folder!**

## How to Use

1. **Browse by category** - Find what you need
2. **Check license** - Each folder has license info
3. **Attribute properly** - Give credit when required
4. **Use freely** - All images are free to use

## Contributing Images

To add images:
1. Find free images (public domain or CC licensed)
2. Place in appropriate folder
3. Add license information
4. Create README with attribution
