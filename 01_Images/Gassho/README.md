# Gassho Hand Position Images

Images showing the Gassho (prayer position) hand gesture.

## What is Gassho?

**Gassho** is the prayer position with hands together at heart center, fingers pointing upward. It's a fundamental Reiki practice position.

## How to Use These Images

- **Teaching:** Show students the correct position
- **Learning:** Reference for practice
- **Materials:** Use in educational materials
- **Cards:** Include in wisdom cards
