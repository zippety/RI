# How to Download Reiki Symbol Images

## Quick Download Guide

### Step 1: Wikimedia Commons (Best Source)

1. **Go to:** https://commons.wikimedia.org/wiki/Category:Reiki_symbols
2. **Browse:** Available Reiki symbol images
3. **Filter by License:** Public domain (CC0) preferred
4. **Download:** Click symbol → "More details" → "Original file" → Download
5. **Save as:** `[symbol_name].png` (e.g., `cho_ku_rei.png`, `sei_he_ki.png`)
6. **Place here:** `01_Images/Symbols/` folder
7. **Add attribution:** Update README.md with source info

### Common Reiki Symbols

- **Cho Ku Rei** - Power symbol
- **Sei He Ki** - Harmony symbol
- **Hon Sha Ze Sho Nen** - Distance symbol
- **Dai Ko Myo** - Master symbol
- **Raku** - Completion symbol

### Step 2: Create SVG Versions (Optional)

For better scaling:
1. Download SVG files if available
2. Or convert PNG to SVG using tools
3. SVG files scale perfectly at any size

## What to Look For

**Good symbol images show:**
- ✅ Clear, accurate symbol drawing
- ✅ High resolution (for printing)
- ✅ Simple background (or transparent)
- ✅ Correct traditional form
- ✅ Educational quality

**Image Requirements:**
- **Resolution:** At least 1000x1000 pixels
- **Format:** PNG (supports transparency) or SVG
- **Quality:** Vector-based preferred (scales perfectly)
- **License:** Public domain or Creative Commons

## Recommended Sources

### Wikimedia Commons
- **URL:** https://commons.wikimedia.org/wiki/Category:Reiki_symbols
- **Best for:** Public domain, traditional symbols
- **License:** Usually CC0 or CC BY

### Create Your Own
- Use public domain symbol designs
- Create SVG versions (vector graphics)
- Ensure accuracy to traditional forms
